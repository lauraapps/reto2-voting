## Pruebas locales
En la terminal o en postman colocar el curl.

El campo vendedor permitido es entre 1 y 10 
El campo producto permitido es entre 1 y 5

```
curl --location 'http://0.0.0.0:3000/consulta?vendedor=1&producto=3'
```
